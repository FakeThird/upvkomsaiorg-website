# About `upvkomsaiorg-website`
**`upvkomsaiorg-website`** is an updated base code for the organization website of the UPV Komsai.Org, the computer science organization based on the University of the Philippines Visayas.

# Instructions and Reminders

Clone the repo and go to your respective page's branch using git checkout. 
(Note: your branch is the same as the name of the page you will be working with like home, events, etc. ).

Once you are in your branch,  go to the folder of the page you will be working with. DO NOT touch other group's folders or push/pull from other branches.

In merging to main, create a pull request explaining the tasks you have done. This will be then assessed by the project manager and merged if the code seems fit. Otherwise, work on the project manager's comments and submit another pull request.
